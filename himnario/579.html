Un hombre de noche llegose a Jesús,

Buscando la senda de vida y de luz;

Y Cristo le dijo: si a Dios quieres ver,

Tendrás que renacer.

CORO

// Tendrás que renacer // // de cierto //

Te digo a ti: tendrás que renacer.

2

Si acaso a los cielos tú quieres entrar,

Y allí con los santos poder descansar;

Si quieres la vida eterna obtener,

Tendrás que renacer.

CORO

// Tendrás que renacer // // de cierto //

Te digo a ti: tendrás que renacer.

3

Amigo, no dejes jamás desechar,

Palabras que Cristo dignose hablar;

Si tú alma no quieres llegar a perder,

Tendrás que renacer.

CORO

// Tendrás que renacer // // de cierto //

Te digo a ti: tendrás que renacer.

4

Hermanos se han ido con Cristo a morar,

A quienes un día querrás encontrar,

Pues esta proclama hoy debes creer,

Tendrás que renacer.

CORO

// Tendrás que renacer // // de cierto //

Te digo a ti: tendrás que renacer.
