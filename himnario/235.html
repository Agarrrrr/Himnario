Gozándome yo voy para aquél hogar,// caminando //

Yo siempre no me quiero aquí quedar,

Caminando para aquel hogar.

CORO

// Caminando // para aquel hogar, donde está Jesús,

// Caminando // de la mano del Señor Jesús.

2

Dejando yo este mundo con gozo voy, // caminando//

Jesús ha prometido conmigo estar,

Caminando para aquel hogar.

CORO

// Caminando // para aquel hogar, donde está Jesús,

// Caminando // de la mano del Señor Jesús.

3

Yo quiero pecadores también llevar, // caminando //

Pues Cristo quiere a todos aquí salvar,

Caminando para aquel hogar.

CORO

// Caminando // para aquel hogar, donde está Jesús,

// Caminando // de la mano del Señor Jesús.

4

Entonces yo a Cristo feliz veré, //en la gloria//

Por siempre con los santos te alabaré,

En aquél hermoso y bello hogar.

CORO

// Caminando // para aquel hogar, donde está Jesús,

// Caminando // de la mano del Señor Jesús.
