Regocijaos los que en el Señor estáis,

Marchemos en pos de Él;

Sirviéndole con fidelidad,

// Hasta llegar a Sión //

CORO

// A Sión // vamos marchando ya,

A esa ciudad de gozo, donde mora Jesús.

2

Alentaos y sin temor marchad,

Dejando angustia y todo afán;

No viendo a diestra sino a Él,

// Hasta llegar a Sión //

CORO

// A Sión // vamos marchando ya,

A esa ciudad de gozo, donde mora Jesús.

3

Ya no temed, Cristo al frente va,

Nos librará muy bien del mal;

Pues Él es nuestro Capitán,

// Hasta llegar a Sión //

CORO

// A Sión // vamos marchando ya,

A esa ciudad de gozo, donde mora Jesús.

4

Aleluya y gloria al Rey de Sión,

Va con su pueblo a reinar;

Y ya no habrá llanto ni dolor,

// En esa santa Sión //

CORO

// A Sión // vamos marchando ya,

A esa ciudad de gozo, donde mora Jesús.
