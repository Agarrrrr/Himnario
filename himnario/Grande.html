Grande alegría y contentamiento,

Hay en mi alma por el Señor,

// Es que recuerdo que fui salvado,

Por el enviado que Dios mandó //


CORO

En esta fecha conmemoramos

Cuando su nombre Dios le cambió,

// Pues él oía que le decía:

Ahora tu nombre será Aarón //


2

Fue una mañana muy memorable

Cuando escuchara la voz de Dios,

// Y hoy sus hijos lo recordamos,

Para gozarnos en el Señor //


CORO

En esta fecha conmemoramos

Cuando su nombre Dios le cambió,

// Pues él oía que le decía:

Ahora tu nombre será Aarón //


3

Mucho sufriera toda su vida,

Por enseñarnos la salvación,

// Pero su Dios que le levantara,

Siempre le dio confortación //


CORO

En esta fecha conmemoramos

Cuando su nombre Dios le cambió,

// Pues él oía que le decía:

Ahora tu nombre será Aarón //
