En una cruz de cruel dolor,

Pendiente a Cristo contemplé;

Perdido entonces me miré, // divino amor //

CORO

// Divino amor // tu gran poder me conquistó,

Será de ti mi corazón // divino amor //

2

Tú me miraste, buen Señor,

Ardiendo en santa compasión,

Tu acento fue de salvación, // divino amor //

CORO

// Divino amor // tu gran poder me conquistó,

Será de ti mi corazón // divino amor //

3

Pródigo anduve en mal error,

Del dulce hogar, errante fui,

Tu eterno amor fue en pos de mí, // divino amor //

CORO

// Divino amor // tu gran poder me conquistó,

Será de ti mi corazón // divino amor //

4

No más rebelde, mi Señor,

Ni vil esclavo habré de ser,

Acógeme en tu gran poder, // divino amor //

CORO

// Divino amor // tu gran poder me conquistó,

Será de ti mi corazón // divino amor //
